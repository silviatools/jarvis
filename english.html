Google Sheets — Setup

Steps at Google Cloud Console:

  1. Enable Google Sheets API (APIs & Services → Library)
  2. Credentials → API key — for reading. API restrictions → Sheets API only
  3. Credentials → OAuth client ID — for writing. Type: Web application. In Authorized JavaScript origins add http://localhost:8000 (or whichever port you use)
  4. Open via local server: python3 -m http.server 8000 in the folder with the file
  5. Spreadsheet: Share → Anyone with link → Viewer
ID found in the URL: docs.google.com/spreadsheets/d/ID/edit
